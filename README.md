# Dual-Axis Solar PV Tracking System

## Overview

This project focuses on the development of a dual-axis solar PV tracking system designed to maximize solar irradiance capture throughout varying daylight conditions.

The prototype was originally developed in 2023 using Arduino Nano, LDR sensors, and automated motor actuation. The project was later expanded with simulation models, tracking analysis, and structured technical documentation.

---

## Objectives

- Improve solar irradiance capture
- Dynamically align PV panels with sunlight
- Compare tracking performance with fixed-tilt systems
- Analyze azimuth and elevation tracking behavior

---
